1. 从已经安装过oracle的server或client 拷贝如下文件

```
classes12.jar
oci.dll
ocijdbc10.dll
ociw32.dll
ojdbc14.jar
orannzsbb10.dll
oraocci10.dll
oraociei10.dll
sqlnet.ora
tnsnames.ora
```

2. 配置环境变量

```
Path = D:\oraclient;
TNS_ADMIN = D:\oraclient;
NLS_LANG = SIMPLIFIED CHINESE_CHINA.ZHS16GBK
```

提供个下载连接[10.2.0.1_32bit](http://pan.baidu.com/s/1o69cMYA)
