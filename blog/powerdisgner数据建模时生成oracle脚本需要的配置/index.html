PowerDisgner数据建模时生成oracle脚本需要的配置
ad
Oct 9, 2014
1 minute read

用PowerDisgner12为oracle进行数据建模时，生成的脚本默认是以“区分大小写，生成脚本如下：

...
create table "table_name"
...

这样执行脚本后，表名实际为"table_name"而不是table_name，结果显示不是我们所要的。

于是需要做如下更改：

1. 打开PowerDesiger中
2. physical data model -> Database -> Edit current DBMS
3. Script->Sql->Format
4. CaseSensitivityUsingQuote
5. Check Yes
6. Uppercase only
7. Check Yes
